# 🔧 Исправления и обновления

## Проблема: `sudo: ./mini_switch: command not found`

**Причина**: Скрипт искал бинарник в текущей директории, а он находится в `bin/`.

**Решение**: Обновлен `scripts/demo.sh`:

```bash
# Было:
sudo ./mini_switch veth_sw1 veth_sw2

# Стало:
sudo ./bin/mini_switch veth_sw1 veth_sw2
```

## 🚀 Новые возможности

### 1. Автоматическая проверка бинарника
```bash
# Скрипт теперь автоматически соберет проект если бинарник отсутствует
if [ ! -f "./bin/mini_switch" ]; then
    echo "Error: mini_switch binary not found. Building first..."
    make
fi
```

### 2. Проверка запуска свитча
```bash
# Проверяем что свитч успешно запустился
if ! kill -0 $SW_PID 2>/dev/null; then
    echo "Error: Mini-switch failed to start or crashed immediately"
    exit 1
fi
```

### 3. Быстрый тест (`make test-demo`)
```bash
# Новый скрипт для быстрого тестирования без интерактивности
make test-demo
```
