Mini SONiC

A simplified Network Operating System inspired by SONiC with C++20 modules

System Overview

Protocol-oriented networking stack with C++20 modules. Implements control/data plane separation, SAI abstraction, L2/L3 forwarding, and visualization components.

Components

  • Control Plane / Data Plane Separation
  • SAI (Switch Abstraction Interface) Layer
  • L2 Switching with MAC learning
  • L3 Routing with Longest Prefix Match (LPM)
  • C++20 Modules
  • Cross-platform support (Windows/Unix)
  • Qt6 Visualizer with Boost.Asio
  • Semantic type aliases

Component Architecture

The system is designed with clear separation between control plane (decision making) and data plane (packet processing), connected through the SAI abstraction layer.

100%

LPM Trie Implementation

  • ✅ Insert: O(prefix_length) time complexity
  • ✅ Lookup: O(32) worst case, typically much less
  • ✅ Memory: Efficient prefix compression
  • ✅ Supports: IPv4 addresses with variable prefix lengths
