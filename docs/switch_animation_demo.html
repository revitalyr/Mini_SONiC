📋 Сценарий демонстрации
Эта демонстрация показывает работу сети коммутаторов Mini SONiC в реальном времени. Пакеты генерируются и проходят через коммутаторы, демонстрируя L2/L3 обработку, MAC learning, IP routing и балансировку нагрузки.

📊 Информация о системе

Mini SONiC использует высокопроизводительную архитектуру с разделением плоскостей управления и данных. Каждый коммутатор обрабатывает пакеты через конвейер L2/L3 сервисов с использованием SAI абстракции.
